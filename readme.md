# 《汐梦之歌》Position工具使用文档

## 1. 软件介绍

《汐梦之歌》Position工具是一个专门用于编辑和生成JSON格式位置数据的工具，主要用于游戏或动画中的位置路径编辑。

### 主要功能
- 可视化编辑多个事件的位置路径
- 支持BPM模式和毫秒模式的时间控制
- 提供多种缓动函数选择
- 支持向量操作（镜像、旋转、平移、缩放）
- 实时预览路径效果
- 一键生成JSON格式输出

## 2. 安装与启动

### 安装
1. 从提供的安装包中解压文件到任意目录
2. 运行 `JSON Position Tool.exe` 启动应用

### 启动
启动应用后，会先显示一个启动动画，然后进入主界面。

## 3. 主界面介绍

### 界面布局
- **左侧面板**：事件列表和编辑区域
- **右侧面板**：JSON输出、播放控制和预览区域

### 顶部按钮
- **全屏**：点击后应用进入全屏模式
- **退出**：退出应用

## 4. 基本操作

### 添加事件
1. 在左侧面板点击 "事件" 标题旁边的 "+ 添加" 按钮
2. 新事件会自动添加到列表末尾

### 编辑事件
1. 选择要编辑的事件
2. 在事件编辑区域修改以下参数：
   - **开始时间**：事件开始的时间点
   - **结束时间**：事件结束的时间点
   - **起点**：事件开始的坐标位置
   - **终点**：事件结束的坐标位置
   - **缓动**：选择位置变化的缓动函数
   - **链接**：勾选后，当前事件的开始时间和起点会自动继承上个事件的结束时间与终点

### 删除事件
1. 选择要删除的事件
2. 点击事件右侧的 "删除" 按钮

## 5. 时间模式

### 毫秒模式
- 时间以毫秒为单位显示和输入
- 适合需要精确时间控制的场景

### BPM模式
- 时间以拍为单位显示和输入
- 可以通过调整BPM值来改变实际时间
- 当BPM值变化时，拍数保持不变，实际时间会相应调整

## 6. 向量操作

### 单个事件操作
1. 选择要操作的事件
2. 在 "向量操作" 区域选择操作类型：
   - **水平镜像**：沿水平轴镜像位置
   - **垂直镜像**：沿垂直轴镜像位置
   - **对角镜像**：沿对角线镜像位置
   - **旋转**：输入旋转角度和旋转中心
   - **平移**：输入X和Y方向的偏移量
   - **缩放**：输入X和Y方向的缩放比例
3. 点击 "执行" 按钮应用操作
4. 执行向量操作后，当前事件和下一个事件的链接会自动取消勾选

### 统一事件操作
1. 点击左侧面板 "事件" 标题旁边的 "统一事件操作" 按钮
2. 在弹出的向量操作弹窗中选择操作类型和参数
3. 点击 "执行" 按钮，操作会应用到所有事件
4. 点击 "取消" 或关闭弹窗则不执行操作

## 7. 预览功能

### 时间节点预览
1. 在右侧预览区域的输入框中输入时间（毫秒或拍，根据当前模式）
2. 点击 "获取时间节点" 按钮
3. 预览画布会显示该时间点的位置坐标
4. 输入框值变化时，滑块会自动同步到对应位置

### 播放预览
1. 点击 "播放" 按钮开始预览
2. 点击 "停止" 按钮停止预览
3. 拖动滑块可以手动调整预览时间

## 8. JSON输出

### 查看JSON
- 右侧面板的 "JSON 文本编辑" 区域会实时显示当前编辑的位置数据的JSON格式

### 复制JSON
- 点击 "复制文本" 按钮可以将JSON文本复制到剪贴板

## 9. 常见问题

### 问题：BPM模式下时间计算不准确
**解决方案**：确保BPM值设置正确，系统会根据BPM值自动计算实际时间

### 问题：向量操作后链接被取消
**解决方案**：这是正常现象，因为向量操作可能会改变事件的时间和位置，导致链接关系不再适用

### 问题：预览区域坐标显示不正确
**解决方案**：检查事件的起点和终点设置，确保缓动函数选择正确

## 10. 注意事项

- 请定期保存JSON文本，以防数据丢失
- 大型项目可能会影响性能，建议适当拆分编辑
- 如遇到应用崩溃，请重新启动应用并重新导入JSON数据
